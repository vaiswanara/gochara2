Transit Navigator — Quick Reference

One-page A4 handout — load a Gochara JSON file, pick a date and Janma Rashi to inspect Panchanga and planet transit results.

Quick steps (students)

  1. Open the Transit Navigator app in your browser (open index.html or the hosted URL).
  2. Click Upload json and select a downloaded Gochara JSON file.
  3. Wait for the file to load — the top row will show filename and record count.
  4. Choose your Janma Rashi from the dropdown (required for transit house calculations).
  5. Select the date to inspect. Use ◀ / ▶ and the Step selector to move across dates.
  6. Read the one-line Panchanga summary and the planet transit rows for Phala (Auspicious/In-Auspicious) details.

UI at-a-glance

  • Upload json — load a local JSON dataset (top-level array of records).
  • Janma Rashi — sets the reference house for transit computations.
  • Date picker / Prev / Next — choose or step through dates.
  • Language — switch reference texts (EN / SA / TE / KA).

Tips & common fixes

  • If the app says Could not auto-load JSON, use Upload JSON (browsers block local fetches when opened via file://).
  • Ensure the top-level JSON value is an array of daily objects; nested {"data": [...]} may not be detected reliably.
  • Prefer Date in YYYY-MM-DD for reliable parsing.
